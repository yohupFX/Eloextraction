File Extractor Tool

This tool processes files from multiple folders, renames them based on an Excel mapping, and saves them to a destination folder.

Steps to Use

  1. Download and install Python if it's not already installed.
  2. Clone this repository to your local machine: 
    git clone https://github.com/yourusername/your-repository-name.git
  3. Navigate to the cloned folder: 
    cd your-repository-name
  4. Install required dependencies: 
    pip install -r requirements.txt
  5. Run the script using: 
    python file_extractor.py --source_folders "path_to_folder1" "path_to_folder2" --destination_folder "path_to_destination" --excel_file "path_to_excel.xlsx"

Example Command

python file_extractor.py --source_folders "/Users/YourName/Folder1" "/Users/YourName/Folder2" \
    --destination_folder "/Users/YourName/Destination" \
    --excel_file "/Users/YourName/mapping.xlsx"
